# <b>AutoRia Scraper</b>

## <u><i>Setup & Launch</i></u>
___
To install extensions on Linux, use this commands in terminal: 

- ```cd project_dir/```
- ```pip install -r requirements.txt```

To launch script on Linux, use this commands in terminal:

```python3 run.py``` or ```python run.py```
